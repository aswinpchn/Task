
company-mssql.sql		installation file for MS SQL
company-mysql.sql		installation file for MySql
company-oracle.zip		installation file for Oracle
company-queries.sql	sample queries from the textbook

MS SQL:
1. Install MS SQL Express and SQL Management Studio 
2. Bring up SQL Management Studio to connect to the SQL instance
3. cut & paste entire content from company-mssql.sql into a query window
   and then execute

MySql:
1. Install MySql and workbench
2. Bring up Workbench to connect to MySql
3. cut & paste entire content from company-mysql.sql into a query window
   and then execute

Oracle:
1. Install Oracle
2. unzip company-oracle.zip
3. take a look of README.html and follow the steps.
